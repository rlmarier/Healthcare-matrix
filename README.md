Healthcare Equity Matrix
An interactive 3D framework mapping care enablers against outcome domains, viewed through ACGME competency and equity lenses.
How to use:

Select an ACGME competency tab to change the clinical lens
Select an equity lens to overlay socioeconomic, race and ethnicity, biological sex, or gender identity risk across each cell
Click any cell to explore that intersection further

Developed to support equitable healthcare education and quality improvement.
